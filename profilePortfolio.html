Robot Arms

General description
In this app, two articulated robot arms, one folded back on the other, start in a position of lying along the x-axis. The app shows the solution to the problem of moving the hand at the end of the second arms to whatever target point the user chooses. The default solution involves both arms rotating anticlockwise. But the user can also choose to optimise the solution, where the hand is enabled to reach the target point in the quickest possible time because both arms are each free to rotate anticlockwise or clockwise.

Technical features
  • object-oriented Java code
  • animation through Java timer function
  • Java layout managers
  • Java Graphics (and Graphics2D) class
  • use of Java mathematical (trigonometry) functions
  • arrays
