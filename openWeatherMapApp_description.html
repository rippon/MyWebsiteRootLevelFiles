Open Weather Map App!!

click here for Open Weather Map App demo

 This app produces a bar chart of the current temperatures of up to 7 capital cities.

The user start by clicking 'show capital cities selection'.

A button for each capital city then appears in a large table.

The user then chooses and clicks up to 7 cities.

Then the user clicks 'show graph'.

The user can start again by clicking 'reset'.
